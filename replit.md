# Happy 21 Chapter of Your Life - Interactive 3D Book

## Overview
An interactive 3D book webpage created as a special birthday gift. The book features:
- A beautiful 3D book cover with "Happy 21 Chapter of Your Life" and a large golden "21"
- Click-to-open animation that transitions to the book reading experience
- Realistic page-flipping using the StPageFlip library
- Old diary/vintage aesthetic with handwritten-style fonts
- 21 chapters of heartfelt content

## Tech Stack
- **Vite** - Build tool and dev server
- **page-flip** - Library for realistic page-turning effects
- **Vanilla JavaScript** - No framework needed
- **Google Fonts** - Dancing Script, Caveat, Indie Flower for handwritten look

## Project Structure
```
/
├── index.html      # Main HTML with 3D book cover structure
├── style.css       # All styling including 3D effects and vintage paper look
├── main.js         # Book content and page-flip initialization
├── vite.config.js  # Vite configuration
├── package.json    # Dependencies
```

## Features
1. **3D Book Cover** - Floating animation, gold decorations, heart icon
2. **Opening Animation** - Book rotates and fades when clicked
3. **Page Flip** - Click edges or use arrow keys to flip pages
4. **21 Chapters** - Each chapter contains 21 items related to the theme
5. **Vintage Look** - Aged paper texture, handwritten fonts, warm colors
6. **Responsive** - Works on mobile devices

## Running the App
```bash
npm install
npm run dev
```
The app runs on port 5000.
